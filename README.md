# 🚀 Vsharing

**Vsharing** هو تطبيق متطور وسريع مبني بلغة `C` ومكتبة `GTK4`، يتيح لك مشاركة الملفات ومزامنة الحافظة (Clipboard) بين حاسوبك (Linux) وهاتفك المحمول بسهولة فائقة عبر شبكتك المحلية (Wi-Fi). وداعاً لتعقيدات الكابلات وتطبيقات الطرف الثالث المزعجة!

---

## ✨ المميزات الأساسية (Features)

### 📁 نقل الملفات السريع (File Transfer)
- **نقل ثنائي الاتجاه:** إرسال الملفات من الحاسوب إلى الهاتف، واستلام الملفات من الهاتف إلى الحاسوب بسلاسة.
- **السحب والإفلات (Drag & Drop):** واجهة سلسة تدعم سحب الملفات وإفلاتها مباشرة لسرعة الإرسال.
- **واجهة ويب تفاعلية:** يتم التخاطب بين الهاتف والحاسوب عبر واجهة ويب حديثة وسريعة الاستجابة تفتح في متصفح الهاتف دون الحاجة لتثبيت أي تطبيق إضافي.

### 📋 مزامنة الحافظة الفورية (Clipboard Sync)
- **مزامنة لحظية (Real-time):** انسخ أي نص في حاسوبك، وسيصبح جاهزاً في هاتفك فوراً. اكتب نصاً في واجهة الهاتف، وسيتم نسخه تلقائياً إلى حافظة نظام حاسوبك!
- **ذكاء التتبع:** نظام تتبع ذكي يمنع الارتداد وتكرار النصوص (Ping-Pong Loop) باستخدام ذاكرة حالة موحدة.
- **دعم أنظمة Wayland و X11:** يتجاوز قيود أمان الحافظة في واجهات لينكس الحديثة بفضل التكامل الإجباري والمنسق مع أدوات النظام `wl-copy` و `xclip`.

### 📡 الاكتشاف التلقائي والرادار (Auto Discovery)
- **بروتوكول mDNS/Avahi:** لا حاجة لكتابة عناوين الـ IP يدوياً! يقوم "الرادار" باكتشاف جميع الأجهزة المتصلة وعرضها لك في الواجهة مباشرة.
- **اتصال بمسحة QR:** قم بمسح الـ QR Code الظاهر في شاشة حاسوبك بهاتفك لتبدأ المزامنة في ثانية واحدة.

### 🔔 إشعارات النظام الحية (System Notifications)
- **تكامل D-Bus:** دمج الإشعارات مباشرة مع نظام لينكس عبر بروتوكول D-Bus لتجربة أصلية (Native).
- **عرض نسبة التقدم:** الإشعارات تُحدث نفسها بشكل حيّ لعرض النسبة المئوية (Progress Percentage) أثناء الإرسال والاستلام.

### 🎨 تصميم عصري وبسيط (Modern GTK4 UI)
- **واجهة أنيقة:** تصميم مستقر ومريح للعين مبني على GTK4 الحديثة، مع أزرار تفاعلية وشكل يناسب بيئة عمل لينكس.

## 🛠️ البناء والتشغيل (Build & Run)

المشروع يعتمد على نظام `meson` و `ninja` للبناء السريع. تأكد من توفر المكتبات والاعتماديات الأساسية في نظامك:
- `gtk4`
- `libsoup-3.0`
- `avahi-client` و `avahi-glib`
- `libqrencode`
- `wl-clipboard` (لأنظمة Wayland) أو `xclip` (لأنظمة X11)

### 1. تجهيز بيئة البناء
```bash
meson setup build
meson compile -C build
```

### 2. التشغيل
للاستمتاع بالتجربة كاملة، تحتاج لتشغيل الخادم الخلفي وواجهة المستخدم:

**الخادم (Daemon):**
```bash
./build/vsharingd
```

**واجهة المستخدم (UI):**
```bash
./build/vsharing-ui
```
